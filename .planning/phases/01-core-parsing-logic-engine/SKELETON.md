# Walking Skeleton — مدير ملفات المحاكم (Mahkama Dossier Manager)

**Phase:** 1
**Generated:** 2026-07-06

## Capability Proven End-to-End

يمكن للمستخدم إدخال مسار مجلد محلي يحتوي على ملف Excel لملفات المحاكم، ليقوم النظام بقراءة بيانات الملف وإعادة تسميته تلقائياً على القرص إلى صيغة `السنة_الرمز.xlsx` وعرض قائمة الملفات المحسوبة آجالها مرتبة على صفحة ويب محلية.

## Architectural Decisions

| Decision | Choice | Rationale |
|---|---|---|
| Framework | Python Flask 3.0 | إطار عمل صغير، سهل التشغيل محلياً دون تعقيدات، ومناسب للتحكم في الملفات والقرص الصلب. |
| Data layer | Excel Files + JSON Config | يتم استهلاك البيانات مباشرة من ملفات Excel المخزنة محلياً، وحفظ الإعدادات الخاصة بمدد الآجال في ملف JSON محلي. |
| UI | Vanilla HTML5 / CSS3 / JS | بناء واجهة خفيفة تدعم RTL بالكامل، وتعتمد على هوية بوابة محاكم (الأزرق والذهبي) والخطوط المحلية دون الحاجة لمكتبات خارجية. |
| Deployment | Local server run | تشغيل خادم محلي على جهاز المستخدم لضمان سرية البيانات الكاملة والعمل بدون إنترنت. |
