## تصميم لوحة التحكم (Layout & Styling)

| Option | Description | Selected |
|--------|-------------|----------|
| هيكل تقليدي علوي (Top Navigation) | شريط علوي مستوحى من بوابة محاكم، يليه شريط اختيار المجلد، وجدول الملفات بكامل العرض مع نافذة منبثقة (Modal) للإعدادات | ✓ |
| تقسيم طولي (Sidebar on Right) | شريط جانبي على اليمين لاختيار المجلد والوصول السريع للإعدادات، والجدول الرئيسي في اليسار لجسم الصفحة | |
| نظام علامات التبويب (Tabbed Layout) | تبويب أول للوحة التحكم والجدول، وتبويب ثانٍ مستقل لإعدادات الآجال والمدد للقضايا | |

**User's choice:** هيكل تقليدي علوي (Top Navigation)

## إبراز مستويات الاستعجال في الجدول

| Option | Description | Selected |
|--------|-------------|----------|
| جدول كلاسيكي ببطاقات ملونة (Status Badge) | جدول قياسي بخلفية محايدة مع عمود لبطاقة الحالة ملونة (أحمر، برتقالي، أخضر) لتسهيل الفرز البصري | |
| خلفية صفوف ملونة بالكامل (Color-coded Rows) | تلوين صف الملف بأكمله بلون الاستعجال خفيفاً (أحمر فاتح جداً، برتقالي فاتح، أخضر فاتح) للتمييز الفوري للأولوية | ✓ |
| مؤشر جانبي بسيط (Minimalist Border/Dot) | نقطة دائرية ملونة أو خط جانبي ملون بجانب رمز الملف دون تغيير ألوان الجدول | |

**User's choice:** خلفية صفوف ملونة بالكامل (Color-coded Rows)

## واجهة اختيار المجلد والبحث

| Option | Description | Selected |
|--------|-------------|----------|
| حقل إدخال مسار مباشر (Folder Path Input) | حقل نصي مخصص لكتابة أو لصق مسار المجلد مع زر مسح المجلد عريض باللون الذهبي | |
| قائمة بالمسارات السابقة (Recent Paths History) | حقل مسار مدعوم بقائمة منسدلة بالمسارات التي قام المستخدم بمسحها مؤخراً لتسهيل إعادة الاستخدام | ✓ |
| مدخل مسار مع شريط بحث جانبي لتصفية النتائج | حقل المسار في الأعلى مع خانة بحث مدمجة في شريط الجدول للفرز والتصفية الفورية | |

**User's choice:** قائمة بالمسارات السابقة (Recent Paths History)
