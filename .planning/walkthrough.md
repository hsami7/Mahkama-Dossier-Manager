# Walkthrough — مدير ملفات المحاكم (Mahkama Dossier Manager)

**Phase:** v1 Complete
**Date:** 2026-07-06

## Accomplishments

لقد تم تطوير وتفعيل تطبيق ويب محلي متكامل لإدارة وتتبع ملفات المحاكم المغربية بنجاح. يجمع التطبيق بين البساطة والسرعة عبر العمل بالكامل بدون اتصال بالإنترنت ودون الاعتماد على مكتبات خارجية ثقيلة.

### 1. محرك الباك اند والتحليل (`engine.py` & `app.py`)
- **تحليل Excel بدون تبعيات**: فك ضغط ملفات `.xlsx` وقراءة بيانات صفوف المحكمة من الملفات مباشرة عبر `zipfile` و `xml` في Python.
- **معالجة التواريخ والرموز**: استخراج السنة والقسم وتاريخ التسجيل وتجاوز الصيغ الرقمية الخاصة بـ Excel.
- **حساب الأجل وتصنيف الاستعجال**: طرح الأيام المتبقية ومقارنتها بالحد القانوني للقضية وتصنيفها بألوان (أحمر، برتقالي، أخضر).
- **نظام حفظ وتتبع إنجاز الملفات (مدمج بملف Excel)**: حفظ حالة الملف كـ "مكتمل" مباشرة داخل ملف الـ Excel نفسه (عن طريق إدراج ملف علامة `completed.flag` داخل أرشيف الـ Zip للملف) مما يتيح استمرار حفظ حالة الملف حتى لو تم نقله أو نسخة لجهاز آخر، دون التسبب في تلف محتواه أو تعارض مع برنامج Microsoft Excel.
- **إعادة تسمية ذكية للملفات**: تعديل اسم ملف Excel تلقائياً على القرص إلى `السنة_الرمز.xlsx` مع معالجة الأخطاء الناتجة عن فتح الملفات (File locks) وحمايتها من استبدال البيانات (Overwrites).
- **خادم Flask**: إعداد خادم محلي خفيف يخدم الواجهة ويدير مسارات REST APIs للمسح وحفظ الآجال.

### 2. واجهة مستخدم ويب تفاعلية RTL (`index.html` & `style.css` & `main.js`)
- **تصميم متكامل مع بوابة محاكم**: شريط علوي وهيكل ألوان أزرق وذهبي مطابق للهوية الرسمية، مع دعم كامل للغة العربية واتجاه الكتابة RTL.
- **مستعرض مجلدات محلي ذكي (Visual Local Folder Browser)**: إضافة زر "استعراض..." يفتح نافذة منبثقة تفاعلية للتنقل داخل مجلدات الجهاز واختيار المجلد المطلوب بصرياً، لحل قيود المتصفح الأمنية في الوصول للمسارات الكاملة.
- **تحديد وتصفية الملفات المنجزة (التبويبات)**: تقسيم الجدول لتبويبين ("القضايا الجارية" و "القضايا المنجزة") مع إخفاء المكتملة تلقائياً وتحديث عداداتها لحظياً.
- **تحديد/إلغاء تحديد الكل مجمع**: إضافة خانة اختيار (Checkbox) في رأس الجدول لتحديد كل القضايا الجارية كـ "مكتملة" أو العكس دفعة واحدة بضغطة زر.
- **هيكل الجدول والهيكل الفارغ (Skeleton)**: عرض جدول فارغ ينبض بانتظار الإدخال، والذي يستبدل فوراً بصفوف بيانات ملونة بالكامل عند المسح.
- **إدارة المسارات السابقة**: حفظ المجلدات التي تم مسحها مسبقاً في LocalStorage وإتاحتها في قائمة منسدلة لتسهيل الفرز.
- **نافذة الإعدادات المنبثقة**: نافذة Modal مخصصة لتحديث الآجال القانونية للفئات من 7201 إلى 7215، مع حفظ التحديثات في ملف JSON وتحديث الجدول فوراً.
- **بحث ديناميكي وسريع**: فلترة وتصفية صفوف الجدول فورياً حسب القاضي، الرمز، الأطراف، أو فئة القضية.

## How to Run locally (كيفية التشغيل المحلي)

لتشغيل التطبيق على جهازك المحلي، يرجى تفعيل البيئة الوهمية وتشغيل الخادم:

```bash
# 1. تفعيل البيئة الوهمية (Virtual Environment)
source venv/bin/activate

# 2. تشغيل خادم الويب
python3 app.py
```

بمجرد تشغيل الخادم، افتح المتصفح وتوجه إلى الرابط التالي للبدء:
`http://127.0.0.1:5000`
